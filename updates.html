v1.5 New Logo & Account Persistence Fix Latest Jul 24, 2026
Feature Fix
  • New logo across the board — sign-in screen, token gate, header, and this changelog
  • Accounts were getting wiped on every redeploy. The server started accepting logins before it had finished restoring saved accounts from the Discord channel backup, so a fresh deploy would briefly think no one had ever registered
  • Server now waits for the account restore to finish before it opens up, so your login sticks around across restarts and redeploys
  • Sessions no longer get invalidated on restart when a fixed JWT_SECRET is configured — previously a new one was generated every time the process restarted, signing everyone out
v1.4 DMs, Member Profiles & Delete Messages Jun 23, 2026
Major Feature
  • Member sidebar is here — shows who's online, idle/DND, or offline, with avatars and status dots
  • Click a member to pull up a DM view. Messages come in live, no refreshing needed
  • You can actually send DMs now, straight from the dashboard
  • Hover a message to get a trash icon — delete works on channel messages and DMs the bot sent
  • Added a back button on the DM view so you're not stuck there
  • New routes on the backend for members, DMs, and message deletion
  • Turned on member + presence intents so status actually updates live
v1.3 Theme Overhaul Jun 23, 2026

v1.2 Custom Bots & Multi-Bot Sending Jun 20, 2026
Feature Fix

  • Sidebar for adding/removing custom bots, one click and you're done

v1.1 Reply System, Notifications & Live Feed Jun 15, 2026
Feature Fix Tweak
